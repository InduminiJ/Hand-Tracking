# Hand-Tracking

## Prerequisites
Needs `python`, `opencv`, `numpy`, `mediapipe`

`$ pip install opencv-python`

`$ pip install numpy`

`$ pip install mediapipe`

## Installation/Usage
cd into the Hand-Tracking directory:

`$ cd /Path/to/Hand-Tracking`

run the program:

`$ python handTracking_GoodBad.py`
